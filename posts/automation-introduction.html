Introduction

Automation is an act/technique or technology used when making a process more automatic than before, usually with minimal human assistance.

Automation applied to an inefficient operation will magnify the inefficiency! - Bill Gates

Automatic systems are part of our daily lives, you can probably find atleast one in your home that is man-made! Hint: boiler. Without this invention how would you know when the water temperature is just right? I guess you would have to touch water with your bare hand now and then and turn boiler on and off based on that and in the end you would probably still make an error. This less efficient system is called non automatic system. Classifcation of systems in automation is explained in the later section.

But what about automatic systems that are not man made? Are there any? Yes! Nature knew well before us that automation is the way to go and our bodies are the very proof! Every cell, organ that is part of body needs to have temperature inside well defined limits, it is duty of one complex system (body) to regulate that without any outside intervention.

What exactly is system?

System is an organized separated set of interconnected elements forming some functional whole, with specific goal in mind.

We could talk about all different kinds of systems(Biological, Social, Techical, Economical...), but most notable are: Physical System and Mathematical System. Key difference between these two systems is that they operate on different elements that are part of different sets - space. We can define them like so:

  • Physical System is a system where elements are substance and energy. They form a set which is said to be part of physical space.
  • Mathematical System is a system where elements are connected by certain axioms. They form a set which is said to be part of mathematical space.
  • Organized Physical System is a collection of subsystems interconnected in some functional whole, with specific goal in mind. This is achieved with information/energy or matter exchange between subsystems.

Most important characterstics of the system is his structure. Structure of the system covers all his subsystems with all their inner and outer connections.

Model of the physical system is an idealized, fictive system, that retains properties of the real system important for its analysis.

Mathematical model of the physical system describes relation between input and output in a formal mathematical way.

System state

State of the system represents internal configuration/properties of its structure at given time and is a result of the inital state of the system and corresponding applied inputs.

State of the system could be deterministic or non deterministic - stochastic. In determinstic systems state would always be the same at given point for the same given set of inputs and initial coondition, while this is not true for stochastic systems.

Inputs of the system are external substantial effects that affect given system. They are used to change the state of the system.

$$x_{input}=\begin{bmatrix}x_1\\x_2\\\vdots\\x_n\end{bmatrix}=x_i(t)$$

Outputs of the system are the results of the given system for which we are interested. Output is affected by the input and the initial state of the system.

$$x_{output}=\begin{bmatrix}x_1\\x_2\\\vdots\\x_n\end{bmatrix}=x_o(t)$$

System response is time dependence between input and output.

Signals

In automation signals are used to enable communication between systems and their elements. Communication system must have transmitter, channel, receiver, processing unit. Where processing unit is optional. Signals that try to disturb the system from achieving desired result are called Disturbances of the system. They usually occur outside of the system and are stochastic in nature.

Automation

Not even best constructed airplanes can fly themselves alone. In order for a plane to fly towards the desired destination some sort of control is required(ie. autopilot, pilot).

Well airplanes are made of really complex systems, surely there is a simpler system which can achieve desired results without outside control? -Short answer: No! Try to think of any such system. Boiler!!! -Boiler is already a system that contains control subsystem. If you remove control subsystem boiler wouldn't do anything or you wouldn't get desired results.

Put simply: You can't expect object to behave like you desire without some sort of control over that object.

So what exactly is the object? Object in automation represents system that we wish to control in order to get desired results. Such objects require control system and these objects are called controlled objects. The purpose of control system is to control the object in such way so that the controlled object behaviour is close as possible to desired. System that is responsible for updating/changing controlled object behaviour is called processing unit.

Systems that contain control part and their respective processing unit are called control systems while systems that contain control system and their respective controlled object are called automatic system.

Types of automatic systems

  1. Non-automatic system - system in which human is the only control system
  2. Semi-automatic system - system in which huamn is the part of the control system
  3. Automatic system - human is not port of the control system

Open automatic system

Main assumption of this system is that disturbances are not chnaging over time, ie. they are constant. Thus input is just formed based on desired output.

$$x_{input}=x_{input}(x_{output})$$

Disturbances in some cases could be measured and compensated for, in which case the system is called open automatic system with compensation.

$$x_{input}=x_{input}(x_{output}, x_{disturbance})$$

In cases of ideal measurment of disturbance, ideal behaviour of systems that compensate for that error is achievable.

Closed automatic system

Disturbances in these systems are allowed to change with time. Thus input is formed based on error in output.

$$x_{input}=x_{input}(x_{desired output}-x_{output})$$

Since in these systems we wait for error to occur before changing input, ideal behaviour of such systems is not possible.

Control system can guarantee error compensation only in cases where that error is well below threshold.

Open and closed automatic system

Input for such automatic system is formed by measuring and compensating for disturbances and error in object behaviour.Both direct and indirect compensation is included in this system and thanks to indirect compensation even disturbances that are not measured by direct compensation are accounted for while making a system more stable then before.

$$x_{input}=x_{input}(x_{desired output}-x_{output}, x_{disturbance})$$

This combined automatic system is much more complex and stable then just open or closed automatic system.
