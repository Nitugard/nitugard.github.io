ODE

2020-12-12
Ordinary differential equation(ODE) is a differential equation containing one or more functions of one independent variable and the derivatives of those functions.

Introduction

Ordinary differential equation - ODE. Contrary to ODE's are partial differential equations - PDE's. Difference between ordinary and partial:

  1. ODE's are easier to solve
  2. ODE's are simpler to understand
  3. ODE's have one independent variable where as partial have more
  4. ODE's can only be used to model simple physics phenomenas, ie. spring motion

ODE is usually in the form: $$ F(x, y, y',y'',y''',\dots , y^{(n)})=0 $$

Solution to ODE can be found in multiple forms:

  1. explicit form: \( y=f(x,C_1,C_2,\dots,C_n) \)
  2. implicit form: \( f(x,y,C_1,C_2,\dots,C_n)=0\)
  3. parametric form: \( x=g(t,C_1,C_2,\dots,C_n),\ y=f(t, C_1, C_2,\dots,C_n) \)

These solutions represent a family of functions. Particular solution is found if we know the inital conditions, ie. in the case of oscilating spring that could be initial position. These are called initial value problems - IVP.

For some types of ODE's, given IVP, we may use certain theorems that tell whether given IVP has solution, and if so whether it is unique. This is very important as we may not be able to solve differential equation but we would like to know whether there exsits solution for our initial values.

Given a solution one can easily find differential equation by expressing constants through derivatives of the function.

Autonomous ODE's

Differential equation is said to be autonomous if it is given in the form that does not depend(explicitly) on the independent variable. If the independent variable is time, such autonomous equation differential eq. is also called time-invariant.

Differential equations that model real physical systems usually have independent variable set to time, ie. spring oscilates as time progresses and are expressed as autonomous systems. If, on the other hand, such systems depended on time explicitly, then that would suggest that such system is time dependent, physics laws would seem to change depending on the time.

$$ F(y, y', y'', \dots, y^{(n)})=0,\ y=f(t)$$
