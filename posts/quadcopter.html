The Quadcopter Project

2017-06-09

Hobby project

Quadcopter

I spent twenty years bound to earth, walking, running and walking some more. Eventually I got tired. I sat down and decided to build something that won't share the same faith! The quadcopter project was born. Atleast this is my part of the story, so bare with me.

The Quadcopter Project was not cheap, had lots of beginner mistakes, but the end result was worth it!

Flying 101

Having no prior experience this project was nearly destined to fail. Alas, I had faith and motivation which gave me the strength to overcome every mistake I made. But before I allowed myself to make mistakes, I spent few weeks researching this topic, since like I said I had no prior experience in anything related to flying. Note that this project was finished while I was still in the high school!

Getting the parts that eventually fit together was not that difficult thanks to the global standardization of the manufactoring process, so even a teenger could do that! The toughest part was balancing the budget and choosing the most optimal parts - this is where I spent my time the most while researching.

Different combinations of motors, propellers, batteries and frames would yield different flight times. If you add camera and other fancy equipment into the mix the flight times would drop down drastically mostly proportional to the weight added. Supporting the newly added weight meant stronger motors, bigger propelers which implied larger battery to support it and again this implied that the frame size eventually would have to increase as well as the cost! This leads us to the next topic:

The law of diminishing return

The law of diminishing return states that in the production system beyond some point the output will start to decreasae as we increase input.

This law while mostly applied in economics could also be applied in our case. While trying for example to get the highest flight times possible while we only vary one part, ie. propellers. At some point buying larger propellers will result in lower flight times! Bigger is not always better.

The key here is knowing when to stop.

Parts

  1. TAROT 650 Carbon Fiber Frame x 1
  2. T-Motors x 4
  3. Turnigy Graphene x 1
  4. Flight Controller x 1

Putting it all together

Yay! The parts have arrived after few months of waiting. Now all the fun can begin. Connecting the cables, soldering, and searching for that one missing screw are some of the stuff you'll find yourself enjoying (with a bit of sarcasm).

If you got the right parts, as I somehow did on the first try. Then this process is quite easy. Unless...

You have that luck to accidently short circuit some of the components multiple times as you try to do some weight cutting.

Result

The test run lasted for about ~28 minutes, this is without camera and with GPS. Satisfaction and experience I got while working on this project is unmesuarable (probably because there is no way to properly quantify it).
