Introduction

In order to efficiently analayze some physical process or behaviour it is neccesary to describe them using math. Math is based on equations that contain already known laws from physics, chemistry, biology...Based on these laws we construct a differential equation that most closesly ressembles behaviour or the process of the system.

Systems analayzed in this article are time dependent(ie. their state depends on time), and as such are described using differential equations where the time is the only independent variable. Example of such system is some object attached to a spring in a fluid as the location(speed) of the object depends only on time.

Systems that also depend upon the location in space are described using partial differential equations where the time and coordinates are independent variables. Partial differential equations are much more complex to solve for, hence no partial differential equation will be used throughout article. Example of such system would be temperature in a room.

In this article we are only concerned with systems that have one input and one ouput. In most cases such systems could be described by differential equation in the form:

$$\phi(t,x_{out}, \dot{x_{out}}, \dots, x_{out}^{(n)}, x_{in}, \dot{x_{in}}, \dots, x_{in}^{(m)} = 0, m\leq n$$

In general form this equation represents system that is nonlinear and nonstationary. Important condition put is that m has to be less or equal to n, if this was not the case system would have an ability to predict.

Stationary system or Time invariant system is system for which time delay on the input directly equates to time delay of the output. Relationship between input and output does not depend on time, and is constant.

$$[x_{out}(t)=x_{out}(x_{in}(t), t)=x_{out}(x_{in}(t))$$

Linear system is a system that has additive and homogenous properties. Changing the input in a linear way will change the output in the linear way.

Linear Time Invariant(LTI) system is linear and time invariant and as such every LTI system can be described by a single function called its impulse response.
